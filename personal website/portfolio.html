Banking System

This project involves the use of multithreaded servers in Java to handle packets from clients (including bank tellers, ATMs, and customers) and includes a GUI. It has basic banking features like depositing and withdrawing money, self-serve and bank teller options, and uses a HashMap to store Customer objects. In a team of 5, my roles included creating diagrams, flowcharts, and the GUI, implementing button functions, and assisting with the server development.
